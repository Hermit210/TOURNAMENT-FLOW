Documentation

QuestFlow automates rewards using a backendless Kwala workflow that listens to on-chain events and executes minting + notifications.

  1. Deploy QuestController and EpicNFT to Base Sepolia.
  2. Grant minter to the Kwala signer on EpicNFT.
  3. Fill addresses in kwala/kwala-nft-reward.yaml and secrets.
  4. Start the Kwala workflow.
  5. Use Trigger page to emit the event.

For production, connect Rewards to your webhook sink or an indexer (e.g., The Graph) for live activity.
