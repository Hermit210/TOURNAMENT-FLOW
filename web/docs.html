Documentation

TournamentFlow is a decentralized gaming tournament platform that automates player registration, match scheduling, and prize distribution using Kwala's event-driven serverless workflows.

Quick Start Guide

1

Deploy Tournament Contracts

Deploy TournamentManager and AchievementNFT contracts to Kwala Testnet.

npx hardhat deploy --network kwala-testnet
2

Configure Kwala Workflows

Set up automated tournament management with event-driven workflows.

# kwala/tournament-automation.yaml
name: tournament-automation
triggers:
  - event: TournamentCreated
  - event: PlayerRegistered
  - event: TournamentCompleted
3

Deploy Workflows

Start Kwala workflows to automate tournament lifecycle management.

kwala deploy tournament-automation.yaml
4

Create Your First Tournament

Use the Tournaments page to create and manage gaming tournaments.

Smart Contract API

TournamentManager.sol

function createTournament(string name, uint256 maxPlayers, uint256 entryFee)

Creates a new tournament with specified parameters.

function registerForTournament(uint256 tournamentId, string username) payable

Registers player for tournament with entry fee payment.

AchievementNFT.sol

function mintAchievement(address to, string achievementType, uint256 tournamentId)

Mints achievement NFTs for tournament winners and participants.
